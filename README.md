# BMI計算アプリの説明
-このアプリは身長(m)と体重を入力することでBMI値を表示することができます。  
*身長の入力値はmを基準にしてくださいｃｍで入力すると値がおかしくなります*    


-さらに二人分の値を出力できるように設計しています。  
-このプログラムは一部サイトから引用したものとなっています。
